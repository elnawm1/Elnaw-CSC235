Bootstrap Components Explanation

For this assignment, I added two Bootstrap components to the existing page:

1. Accordion Component
2. Alert Component

Why I chose these components:

Accordion:
I chose the accordion because it helps organize information while without taking up too much space on the page. In lecture 3, we discussed responsive design and making material easy to read. The accordion allows users to select what they wish to learn about (Lectures 1, 2, or 3) while hiding the rest until they need it. This makes the page more organized and less overpowering. I included information from our actual lectures to help students study.

Alert:
I included a dismissible alert at the top of the main content. This is an effective technique to welcome users or provide vital announcements. In lesson 2, we discussed user experience, and the alert communicates with users as soon as they load the page. The fact that it can be closed (dismissible) offers consumers control over their interface, which is a good practice we discovered.

Both components use Bootstrap's built-in JavaScript, which we've linked to on the page, so I don't have to create any difficult code. This builds on what we learnt about leveraging frameworks to speed up development.

The page already has a navbar with dropdowns, so adding these two components increases the number of interactive items without complicating the layout.
